PWA Studio in GitHub

The Magento PWA Studio project is an open source GitHub project that welcomes community contributions and collaboration.

All the tools listed below are available as separate packages in the PWA Studio monorepo.

If you have any questions, please join the #pwa Slack channel at magentocommeng.slack.com.

Venia Storefront source

Project source code for a conceptual and customizable PWA storefront for Magento 2. It is built using the Peregrine component library and Buildpack tools.

Peregrine source

Project source code for the collection of UI components for the Magento PWA projects.

Buildpack source

Project source code for the build and layout optimization tooling for Magento PWA projects.
